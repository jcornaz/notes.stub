lazyParent

fun <ChildType, ParentType> lazyParent(init: () -> ParentType, removeChild: ParentType.() -> Unit, addChild: ParentType.() -> Unit): MutableLazyDelegate<ChildType, ParentType>

Create a delegate for a property parent of the owner class The parent will be created if needed only The parent children list will always be up-to-date

Parameters

init - Will be invoked to create a new instance of the parent if needed

removeChild - Will be invoked if the owner has to be removed from a parent

addChild - Will be invoked if the owner has to be added to a parent
